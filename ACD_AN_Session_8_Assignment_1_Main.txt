a. What is Shared Preferences. 
Android provides many ways of storing data of an application. One of this way is called Shared Preferences. Shared Preferences allows us to save and retrieve data in the form of key,value pair.

In order to use shared preferences , we have to call a method getSharedPreferences() that returns a SharedPreference instance pointing to the file that contains the values of preferences.

Examples include
•Endpoint URLS,
•The security access token,
•Certain user preferences like color, title, language, etc.
-------------------------------------------------------------------------------

b. What values can be saved in SharedPreferences.

The following kind of values can be saved in SharedPreferences :

•Boolean values
•Float values
•Integer values
•Long values
•String values
